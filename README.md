# Electric Scooter Management System

Django web application for managing electric scooters, reservations, rentals, payments, and maintenance.

---

## Features

- User registration and login
- Scooter reservations
- Instant ride system
- Ride history
- Payment simulation
- Scooter tracking
- Maintenance management
- Admin panel
- Zone pricing system

---

## Technologies Used

- Python
- Django
- SQLite
- HTML/CSS
- Bootstrap
